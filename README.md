# Bike Sales Dashboard (Microsoft Excel)

## Overview
This project is an interactive Bike Sales Dashboard built in Microsoft Excel to analyze customer purchasing behavior using Pivot Tables, Pivot Charts, and Slicers. The dashboard provides insights into income, age groups, regions, education, and commuting distance to support data-driven business decisions.

---

## Objective
To transform raw customer data into an interactive dashboard that helps identify sales trends and customer purchasing patterns.

---

## Dataset
- Customer demographic data
- Income
- Marital Status
- Gender
- Education
- Occupation
- Region
- Commute Distance
- Purchased Bike (Yes/No)

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning
- Data Filtering & Sorting

---

## Dashboard Features
- Average Income by Gender and Purchase Status
- Customer Age Bracket Analysis
- Total Bikes Purchased by Region
- Customer Commute Distance Analysis
- Interactive Slicers:
  - Marital Status
  - Region
  - Education
  - Purchased Bike

---

## Key Insights
- Compared average customer income based on bike purchase decisions.
- Analyzed purchasing patterns across different age groups.
- Identified regional distribution of bike purchases.
- Evaluated the relationship between commute distance and purchasing behavior.
- Built an interactive dashboard for quick business analysis.
